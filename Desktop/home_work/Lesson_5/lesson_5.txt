Метод класса наследуются

def self.name_method
  #реализация метода класса
end

class << self
  def name_method
    #реализация метода класса
  end
end
 
Переменная класса доступна и разделяется не только в целом у самого класса,
но и с его наследникамиы

@@val_name = 0

include - включает методы которые есть в модуле как инстанс (как методы зкемпляра)
extend - включает методы которые есть в модуле как методы класса
